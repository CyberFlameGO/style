## Zeit Style
Zeit style is a modern css library, originating from [Zeit Design](https://zeit.co/design).

<br/>

## Usage

- with npm: `npm i @zeit-ui/style`

- with CDN: [jsdelivr cdn](https://cdn.jsdelivr.net/npm/@zeit-ui/style@latest/dist/style.css)

<br/>
